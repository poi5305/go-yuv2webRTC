## Install

```
# mac
brew install libvpx pkg-config

# linux
apt-get install libvpx-dev
```

```
go get github.com/poi5305/go-yuv2webRTC/...
```

```
cd $GOPATH/github.com/poi5305/go-yuv2webRTC
go run main.go
```

Open http://localhost:8000
